## 📝 PROJECTS OVERVIEW
This repository **ms-word-projects** contains the files and folders that shows application of various **MS Word** tools. Each Project is designed to showcase the Professional document standards:
* Heading Styles
* Formatting Options (`bold`, `italic`, `underline`)
* Tables(e.g.,`Weekly Timesheet Log.docx`)
* Cover pages and index (e.g, `Customer Database Report.docx`, `Inventory Tracking Sheet.docx`)
* Every project includes both the original .docx and high-quality PDF version

---

### Project Breakdown
📁 **Data Entry Projects (Folder)**\
A Collection of 9 subfolders focusing on document structure and accuracy.
* Each subfolder contains the source **Word Document** and **PDF Version** of that document.

---

📁 **Invoice Formats (Folder)**\
This folder showcases two distinct professional invoice designs organized by industry. Includes both document and PDF Version.
1. **Computer accessories (Subfolder)**
	* Retail focused layout for hardware sales
2. **IT Services (Subfolder)**
	* Service based format customized for consultancy and technical support.
	
---

📁 **Mail Merge (Folder)**\
This folder describes the usage of Mail Merge and has two subfolders.
1. **Housing Society Notice (Subfolder)**
	* Utilises the **Letter Mail Merge** feature of MS Word to generate 20 individual notices for Housing Society members.
2.  **Coffee Label (Subfolder)**
	* Demonstrate the **Label Mail Merge** feature of MS Word to generate Label for Coffee branding and packaging.
